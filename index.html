Bubble Sort

Bubble Sort ist ein Sortieralgorithmus. Dabei wird ein Array paarweise von links nach rechts in einer Bubble Phase durchlaufen. Array ist eine Eingabeliste. Gestartet wird bei der ersten Zahl, diese wird verglichen mit der Zahl direkt nebenan, demfalls rechts. Wenn die erste Zahl großer ist, bleibt sie stehen. Wenn die erste Zahl kleiner ist, wird sie mit der nächsten Zahl vertauscht. Dannach wird direkt das nächste Paar miteinander verglichen bis die gesamte Liste e0inmal durchlaufen wurde. Dieser Prozess wird so oft wiederholt, bis die Liste sortiert ist.

Beispiel:

1, 4, 7, 6, 2, 5, 8

  • 1 ist kleiner als 4. Es bleibt
  • 4 ist kleiner als 7. Es bleibt
  • 7 ist grösser als 6. Es wird vertauscht
  • 7 ist grösser als 2. Es wird vertauscht
  • 7 ist grösser als 5. Es wird vertauscht
  • 7 ist kleiner als 8. Es bleibt
So sieht die Liste nach dem 1. Durchlauf Sortierverfahren aus --> 1, 4, 6, 2, 5, 7, 8 7 Nach dem 1.Durchlauf macht man so lange Durchläufe bis die Liste sortiert ist.
